Что такое язык TiuTiu?

TiuTiu ‒ это искусственный язык вьетнамцев, который по старой американской легенде вьетконговцы использовали для общения между собой.

Зачем нужен этот сайт?

Этот сайт создан для тех, кто хочет изучать этот прекрасный язык.
